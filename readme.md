# How to run program

### Command Line:
**Program must be run using command
"python ./Shell.py"**

### INPUT:
**The shell expects to find file name with "input.txt" within the same directory as the source code
**

### OUTPUT:
**The program will output a file named "output.txt" with the results**

## SHELL COMMANDS
 - Init Process: in 
 - Create Process: cr
 - Timeout Process: to
 - Request resource: rq
 - Release resource: rl
 - Destroy : de
